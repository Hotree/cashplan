  • 가장 많이 묻는 질문에 대한 답변
    • 잠금화면에서 어떻게 캐시 적립을 받을 수 있나요?

      잠금화면에서 적립 가능한 광고를 본 후, 잠금 해제만으로도 캐시 적립이 가능합니다. 또한 캐시슬라이드 앱 내 ‘바로 적립받기’ 메뉴에서 다양한 광고를 확인하면, 더 많은 캐시를 적립받을 수 있어요!
    • 상단 알림창에 상태 메시지를 제거하고 싶으면 어떻게 하나요?

      상단 알림창의 상태 메시지는

      안드로이드 OS 4.3버전부터 업데이트된 모든 기기에서 실행중인 앱 표시 내용이 나타나도록 해야하는

      새로운 구글 정책에 따라 적용되었습니다.

      안드로이드 OS 4.3버전을 사용하는 모든 기기 이용고객님께 양해 부탁드립니다.

      앱 실행 중 메세지는 일반적인 알림 메세지이기 때문에, 지속적인 데이터 통신이 발생하는 부분은 아닙니다.

      상단 알림으로 인해 스마트폰 사용이 불편하신 경우,

      상단의 캐시슬라이드 알림창을 클릭한 후, 강제중지 버튼 밑의 알림표시 체크박스에서 체크를 해제하시면 됩니다.
    • 구입한 쿠폰을 환불하고 싶으면 어떻게 하나요?

      쿠폰 환불은 캐시슬라이드 고객 센터를 통해 받으실 수 있습니다.

      환불을 원하는 쿠폰의 상세 내역을 적어서, 앱 내 ‘문의하기’로 보내주세요.
    • 광고에 참여했는데 적립이 되지 않으면 어떻게 하나요?(설치형, 실행형 등)

      다운 받은 어플이 있는 화면을 캡쳐하고,

      캐시슬라이드 앱 내 ‘문의하기’에 이미지를 첨부하여 고객센터로 보내 주세요.

      캐시슬라이드 ID 및 참여 내역을 확인 후 재적립 해드리며, 영업일 기준 최대 2일이 소요됩니다.
    • 로그아웃은 어떻게 하나요?

      캐시슬라이드는 별도의 로그아웃 기능이 없어, 어플 삭제 후 재설치를 해주셔야 합니다.

      어플 삭제 후 구글플레이스토어에서 캐시슬라이드를 재설치 해주세요.

      사용하시던 아이디와 비밀번호로 로그인하시면 기존 적립 캐시 그대로 사용 가능합니다.
    • 이메일 주소 변경은 어떻게 하나요?

      캐시슬라이드 가입 시 기입하신 이메일은 앱 내 ‘문의하기’를 통해 변경하실 수 있습니다.

      캐시슬라이드 ID, 변경 전 이메일 주소, 변경을 원하는 이메일 주소와, 출생년도, 변경 사유를 기재하시어 보내주세요.

      확인 후 빠르게 변경해드리겠습니다.
    • 비밀번호를 잊었을 경우 어떻게 하나요?

      비밀번호는 앱 내 우측 상단 ‘+’ -> ‘계정관리’ -> ‘비밀번호 찾기’ 에서 확인 가능합니다.

      캐시슬라이드 가입 시 기입하신 이메일 주소로 초기화된 임시 비밀번호가 발송되며

      개인 정보 보안을 위해 반드시 임시 비밀번호로 로그인 하신 후, 새로운 비밀번호로 변경 부탁드립니다.
    • 비밀번호 변경은 어떻게 하나요?

      비밀번호 변경은 캐시슬라이드 앱 내 우측 상단의 ‘+’ -> ‘계정관리’ -> ‘비밀번호 변경’에서 가능합니다.
  • 서비스 관련 질문에 대한 답변
    • 캐시슬라이드는 어떤 서비스인가요?

      1300만이 선택한 1등 잠금화면, 캐시슬라이드!

      캐시슬라이드는 더욱 편리한 모바일 생활을 위한 잠금화면 어플리케이션입니다.

      간단히 잠금해제 만으로도 적립 혜택을 받으실 수 있으며,

      생활 속 유용한 정보, 최신 뉴스, 트렌드까지 한번에 첫화면에서 편하게 만나보실 수 있습니다.

       
  • 계정 관련 질문에 대한 답변
    • 회원 탈퇴는 어떻게 하나요?

      캐시슬라이드 회원 탈퇴는 앱 내 우측 상단의 ‘+’ -> ‘계정관리’ -> ‘회원탈퇴’ 에서 가능합니다.

      단, 회원 탈퇴 시 모든 정보(개인정보, 적립 캐시 등)는 자동 삭제되어 복구가 불가능하오니,

      탈퇴 전, 다시 한 번 더 고려해주세요.
    • 로그아웃 상태에서 비밀번호 찾기는 어떻게 하나요?

      로그인 화면에서 ‘비밀번호 찾기’를 클릭 후, 캐시슬라이드 아이디와 가입 시 기입하신 이메일 주소를 정확하게 입력해주세요.

      아이디와 이메일 주소 확인 후 임시 비밀번호가 이메일로 발급됩니다.

      임시 비밀번호로 로그인하신 뒤에는, 개인 정보 유출의 우려가 있으니 ‘계정관리’ -> ‘비밀번호 변경’ 에서 꼭 비밀번호를 변경해주세요.
    • 로그아웃은 어떻게 하나요?

  • 적립 관련 질문에 대한 답변
    • 환불을 받아서 재적립된 적립 캐시는 어떻게 확인 하나요?

      캐시슬라이드 앱을 실행하신 후,

      상단의 캐시 금액을 클릭하시면 자세한 적립 내역을 확인하실 수 있습니다.

      환불된 적립 캐시 금액은 ‘이벤트 적립 캐시’에서 확인이 가능합니다.
    • 쿠폰의 사용기간이 지났어요. 어떻게 하나요?

      쿠폰의 사용기간이 만료된 경우, 상품 가격의 70%를 환불해드리고 있습니다.

      캐시슬라이드 앱 내 ‘문의하기’를 통해 고객센터로 문의해주세요.
    • 구입한 쿠폰은 어떻게 사용하나요?

      쿠폰별 사용 방법이 다르며, 상세 사용 설명은 각 쿠폰 하단에 안내되어 있습니다.

      온라인 쿠폰의 경우 안내된 사용 방법을 참고하시어, 사용처에서 쿠폰 번호를 입력하여 사용하세요.

      오프라인 매장에서 사용하는 쿠폰의 경우 물품 결제 시 쿠폰을 보여주시면,

      바코드 확인을 통해 쿠폰을 사용하실 수 있습니다.
    • 구입한 쿠폰은 어디서 확인 가능한가요?

      구입한 쿠폰은 캐시슬라이드 앱 내 ‘쿠폰함’ 에서 확인 가능합니다.

      구입한 쿠폰별 사용기간이 다르니, 사용 전 반드시 확인해주세요.
    • 광고 적립급이 나와 있고 좌우 슬라이드를 했는데도 적립이 되지 않으면 어떻게 하나요?(노출형)

      아래의 경우 캐시가 정상적으로 적립되지 않을 수 있습니다.

      1. 같은 ID로 여러 기기에서 로그인 되어 있는 경우

      2. 데이터 통신이나 WIFI(무선랜)가 사용 불가인 경우

      3. 핸드폰의 시간이 잘못 설정되어 있을 경우

      먼저 위의 세 가지 경우에 해당되지 않는지 확인을 부탁드리고,

      해당 사항이 없는데도 적립이 되지 않는다면

      캐시슬라이드 앱 내의 ‘문의하기’를 통해 고객센터에 문의해주세요.
    • 구입한 쿠폰을 환불하고 싶으면 어떻게 하나요?

      쿠폰 환불은 캐시슬라이드 고객 센터를 통해 받으실 수 있습니다.

      환불을 원하는 쿠폰의 상세 내역을 적어서, 앱 내 ‘문의하기’로 보내주세요.
    • 광고에 참여했는데 적립이 되지 않으면 어떻게 하나요?(설치형, 실행형 등)

      다운 받은 어플이 있는 화면을 캡쳐하고,

      캐시슬라이드 앱 내 ‘문의하기’에 이미지를 첨부하여 고객센터로 보내 주세요.

      캐시슬라이드 ID 및 참여 내역을 확인 후 재적립 해드리며, 영업일 기준 최대 2일이 소요됩니다.
  • 캐시아웃 관련 질문에 대한 답변
    • 휴대폰 요금 감면 신청 후, 감면내역은 어떻게 확인할 수 있나요?

      요금감면 내역은 신청하신 다음달 10일 이후 모바일 또는 지로 명세서에서 확인 가능합니다.

      명세서에서 요금감면이 확인되지 않는 경우,

      해당 통신사의 고객센터에서 아래 문구로 요금 내역 확인 부탁드립니다.

      < 통신사별 차감내역 확인 문구 안내 >

      SKT – 휴대폰 무료 이용권

      KT – 선납요금

      LG U+ – 선납금

      단, 통신사 미납금이 있을 경우 명세서에서 확인이 되지 않는 경우도 있습니다.

       
    • 환급 시 수수료가 있나요?

      기업은행을 제외한 나머지 은행으로 환급 신청 시에는 500원의 수수료가 발생합니다.

      따라서, 기업은행 외에는 환급 시에 수수료를 제외한 금액이 입금됩니다.
    • 환급 받을 계좌를 잘못 입력한 경우에는 어떻게 하나요?

      신청 시 기입하신 계좌번호, 예금주명은 개인 정보 보호를 위해 암호화되어 저장되기 때문에 변경이 불가합니다.

      환급이 정상적으로 진행되지 않은 경우 캐시슬라이드 적립 캐시로 재적립되며,

      앱 내에서 캐시아웃을 다시 신청해주셔야 합니다.
    • 환급 받을 날인데 환급이 안된 이유는 무엇인가요?

      잘못된 계좌 번호, 예금주명(실명이 아닐 시)을 기입하셨을 경우 정상적으로 환급이 되지 않습니다.

      또한 법인 계좌, 가상 계좌의 경우 지급이 불가하니 꼭 확인해주세요.

      환급이 진행되지 않은 경우에는, 다음주 월요일 캐시슬라이드 적립 캐시로 재적립됩니다.

       
    • 캐시아웃은 언제 들어오나요?

      캐시아웃은 적립 캐시 2만원 이상부터 가능합니다.

      금요일 오전 12시 이전에 신청한 건은 다음주 수요일, 그 이후에 신청한 건은 2주 후 수요일에 일괄 지급됩니다.

      (공휴일의 경우 환급일을 별도 공지합니다.)

      환급 내역은 오후 6시 이후에 확인해주세요.
  • 기타 관련 질문에 대한 답변
    • 문의하기로 메일을 보냈는데, 왜 답변이 늦어지나요?

      문의주신 내용은 순서대로 답변을 해드리고 있으나,

      주말/공휴일 이후 등 문의가 많을 경우에는 답변이 다소 지체될 수 있는 점 양해 부탁드립니다.

      보내주신 문의 사항에 대해 영업일 기준 최대 3일 이내 답변드릴 수 있도록 최선을 다하겠습니다.
    • 이벤트 응모 후에 당첨 확인은 어떻게 하나요?

      이벤트 당첨 확인은 앱 내 ‘이벤트 게시판’에서 확인 가능합니다.

      이벤트에 당첨되신 분들께는 캐시슬라이드 가입 시 기입하신 이메일 주소로 별도의 안내 메일이 발송됩니다.
    • 잠금화면에서 광고를 보고 싶지 않을 때는 어떻게 하나요?

      캐시슬라이드 앱 내 우측 상단 ‘+’ 메뉴에서

      ‘첫 화면에서 앱 실행’을 ‘끄기’로 변경하시면 됩니다.

      해당 기능을 통해 광고 노출을 원하지 않는 시간을 별도로 설정하실 수 있습니다.
    • 상단 알림창에 상태 메시지를 제거하고 싶으면 어떻게 하나요?

      상단 알림창의 상태 메시지는

      안드로이드 OS 4.3버전부터 업데이트된 모든 기기에서 실행중인 앱 표시 내용이 나타나도록 해야하는

      새로운 구글 정책에 따라 적용되었습니다.

      안드로이드 OS 4.3버전을 사용하는 모든 기기 이용고객님께 양해 부탁드립니다.

      앱 실행 중 메세지는 일반적인 알림 메세지이기 때문에, 지속적인 데이터 통신이 발생하는 부분은 아닙니다.

      상단 알림으로 인해 스마트폰 사용이 불편하신 경우,

      상단의 캐시슬라이드 알림창을 클릭한 후, 강제중지 버튼 밑의 알림표시 체크박스에서 체크를 해제하시면 됩니다.
